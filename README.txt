How to use:
1. Open index.html in any browser.
2. Edit the "To Be Added" sections with your real wedding date, venue, city, and times.
3. Upload the folder to a web host like Netlify, GitHub Pages, or Railway static hosting.

Important:
- RSVP and pledge forms currently save data only in the visitor's browser (localStorage).
- To collect real online responses, connect the forms to a backend or database.
